# Taxonomia Semantica do Cosmos Midia

Esta pasta documenta a linguagem semantica oficial do Cosmos Brain para fabricas de uniformes.

A taxonomia separa dado bruto de estrutura semantica:

- Dado bruto: transcricao, legenda, roteiro, anuncio, briefing ou comentario.
- Estrutura semantica: segmento, produto, tipo de dor, hook, narrativa, tom emocional, CTA, objetivo comercial, estilo visual e gatilho visual.

Esta camada nao executa IA, embeddings, busca semantica ou recomendacao. Ela apenas define o vocabulario controlado que essas camadas poderao usar no futuro.

Os dados estruturados ficam em `data/taxonomia/`. A documentacao aqui explica o raciocinio por tras de cada grupo.
