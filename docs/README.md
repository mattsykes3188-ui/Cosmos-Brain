# Cosmos Brain Docs

## Brain Consumption Layer

A Brain Consumption Layer fecha o primeiro ciclo operacional do Cosmos:

```text
ingestao
-> governanca
-> inteligencia
-> consumo operacional
```

Ela permite que o Cosmos Midia consulte padroes aprovados do Brain e gere direcoes de conteudo rastreaveis, sem IA autonoma, embeddings ou recommendation engine avancado.

Documento principal:

```text
docs/architecture/BRAIN_CONSUMPTION_LAYER.md
```

## Research Documents Structure

A Research Documents Structure separa documentos humanos completos dos exports estruturados e da biblioteca oficial governada.

Fluxo correto:

```text
research_documents/raw
-> research_documents/processed
-> research_exports
-> data/research_library
```

Documento principal:

```text
docs/architecture/RESEARCH_DOCUMENTS_STRUCTURE.md
```
